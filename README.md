# אפליקציית לימוד איטלקית - Italian Language Learning Platform

אפליקציית ווב מודרנית ומלאה ללימוד שפה איטלקית עם ממשק עברי, מערכת ניהול Admin מלאה, ומנוע שיעורים אינטראקטיבי בסגנון Duolingo.

## ✅ סטטוס: פרויקט מלא ומוכן לשימוש!

הפרויקט הושלם במלואו, נבנה בהצלחה והוא מוכן לפריסה בפרודקשן.

## תכונות עיקריות

### לתלמידים
- **8 שיעורים במבנה פירמידה** - התקדמות הדרגתית ומובנית
- **למידה אינטראקטיבית** עם 5 אופציות לכל משפט:
  - הצגת תרגום לעברית
  - בדיקת הגייה עם זיהוי קולי (Web Speech API)
  - הצגת כתיב נכון עם הדגשת הטעמה
  - כתיבת הערות אישיות
  - מעבר לשורה הבאה
- **מעקב התקדמות** - שמירה אוטומטית וחזרה לנקודת המשך
- **שמירת מצב** - המערכת שומרת את מיקומך המדויק בשיעור גם כשיוצאים מהאתר או ממזערים את הדפדפן (למחשב ומובייל)
- **פתיחה מדורגת** - כל שיעור נפתח רק לאחר השלמת הקודם

### למנהלים
- **ניהול משתמשים** - יצירה, עריכה, חסימה ושינוי תפקידים
- **בניית שיעורים** - ממשק CRUD מלא ליצירת תכנים
- **עריכת שורות** - הוספה ועריכה של שורות INFO/LANG עם תצוגה מקדימה
- **מעקב התקדמות** - דשבורד עם סטטיסטיקות מפורטות

## טכנולוגיות

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Database**: Supabase (PostgreSQL)
- **Authentication**: סיסמאות מוצפנות (bcrypt) + ניהול תפקידים (RBAC)
- **Speech Recognition**: Web Speech API
- **Text Similarity**: Levenshtein distance algorithm

## מסד נתונים

### טבלאות
- **users** - משתמשים (תלמידים ומנהלים)
- **lessons** - שיעורים (0-8)
- **lines** - שורות בשיעורים (INFO/LINK/LANG)
- **notes** - הערות אישיות של תלמידים
- **progress** - מעקב התקדמות תלמידים
- **speech_attempts** - ניסיונות הגייה והציונים

### Row Level Security (RLS)
כל הטבלאות מוגנות ב-RLS:
- תלמידים רואים רק את הנתונים שלהם
- מנהלים רואים ומנהלים את כל הנתונים
- משתמשים לא מחוברים אינם רואים דבר

## הדגשת הטעמה באיטלקית

המערכת תומכת בהדגשה אוטומטית של הטעמה במילים איטלקיות:

### זיהוי אוטומטי
המערכת מזהה אותיות עם אקסנט (á, é, í, ó, ú, à, è, ì, ò, ù) ומדגישה את ההברה הרלוונטית.

### הדגשה ידנית
ניתן להגדיר כלל הדגשה מותאם עם תחביר `[טקסט להדגשה]`:
- דוגמה: `"[par]lare"` תדגיש את "par" במילה "parlare"

### תצוגה לתלמיד
בלחיצה על "הצג כתיב נכון", התלמיד רואה את המשפט עם ההברות המודגשות **בולד וצבע כחול**.

## בדיקת הגייה

המערכת משתמשת ב-Web Speech API לבדיקת הגייה:

1. **הקלטה** - התלמיד מקליט את עצמו דרך המיקרופון
2. **זיהוי** - המערכת ממירה את ההקלטה לטקסט
3. **השוואה** - חישוב דמיון עם Levenshtein distance (כולל נורמליזציה)
4. **ציון** - הצגת ציון 0-100% ואפשרות לנסות שוב

**סף קבלה**: 80% דמיון (ניתן לשינוי)
