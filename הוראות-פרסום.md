### דרך 1: באמצעות Netlify CLI (מומלץ)

#### שלב 1: התחבר ל-Netlify
```bash
export NVM_DIR="$HOME/.nvm" && [ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
netlify login
```
זה יפתח דפדפן - אשר את ההתחברות.

#### שלב 2: פרסם את האפליקציה
```bash
cd "/Users/danielefter/Documents/Tommy project/project"
netlify deploy --prod --dir=dist --site=wonderful-kringle-a84cac
```

אם מקבל שגיאה, נסה קודם:
```bash
netlify link
```
ובחר את האתר: `wonderful-kringle-a84cac`

## אם עדיין יש בעיות

### בעיה: "המערכת עדיין איטית"
**פתרון**: נקה את ה-cache של הדפדפן:
- Chrome: Ctrl+Shift+Delete → בחר "Cached images and files"
- Safari: Develop → Empty Caches
- או פשוט פתח חלון Incognito

### בעיה: "לא רואה את השינויים"
**פתרון**: 
1. וודא שהפרסום הצליח ב-Netlify
2. רענן את הדף עם Ctrl+Shift+R (או Cmd+Shift+R במק)

### בעיה: "שגיאה בטעינת שיעורים"
**פתרון**:
1. בדוק שה-Supabase זמין
2. בדוק את הקונסול לשגיאות (F12)
3. לחץ על כפתור "נסה שוב"
